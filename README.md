# atas-trading-tools

Indicateurs et outils de trading pour la plateforme **ATAS**.  
Développés par **Mobius Performance Consulting**.

---

## Projets disponibles

| Projet | Branche | Description | Statut |
|---|---|---|---|
| ATAS Lecture CSV | [`atas-kde-levels`](../../tree/atas-kde-levels) | Indicateurs ATAS : import de niveaux depuis un fichier CSV, calcul automatique de niveaux KDE, export OHLC | Stable v1.0.1 |

---

## Utilisation

Chaque projet est disponible dans sa propre branche.  
Pour récupérer un projet :

```bash
# Cloner uniquement la branche du projet
git clone --branch atas-kde-levels --single-branch https://github.com/mobius-performance-consulting/atas-trading-tools.git
```

Consulter le `README.md` de chaque branche pour les instructions d'installation et d'utilisation.
